# Agent with Payment

An AI agent with PayLink payment tools built on LangGraph.

## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/paylinkmcp/agent-with-payment.git
cd agent-with-payment
```

### 2. Set up environment variables

```bash
cp .env.example .env
```

Open `.env` and replace with your actual values:

```env
OPENAI_API_KEY=your_openai_api_key
PAYLINK_API_KEY=your_paylink_api_key
```

### 3. Install dependencies

```bash
uv sync
```

### 4. Run the server

```bash
langgraph dev
```
